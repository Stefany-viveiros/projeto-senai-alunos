Sobre o Projeto SENAI-ALUNOS

Quem Somos

Este site foi desenvolvido pelos alunos do SENAI com o objetivo de facilitar o acesso a informações importantes, como carteirinha de estudante, área do perfil, links úteis e muito mais. A ideia é criar um portal simples, funcional e organizado, focado totalmente no aluno.

Nosso objetivo principal foi criar um site funcional, claro e acessível, demonstrando as habilidades adquiridas ao longo do curso. Buscamos também desenvolver competências importantes para o mercado de trabalho, como organização, comunicação e resolução de problemas.

O projeto utilizou tecnologias como HTML para estruturação, CSS para design e layout, JavaScript para interação e Git e GitHub para controle de versão. Essas ferramentas permitiram criar um site moderno, responsivo e alinhado às boas práticas de desenvolvimento
